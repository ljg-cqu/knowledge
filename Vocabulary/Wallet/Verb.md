# Extract Verbs for Vocabulary Learning

**Task**: Extract verbs (action words, phrasal verbs, verb phrases) that are essential for language mastery from the source text about **Wallet**.

**Definition**: Each entry focuses on one verb or phrasal verb with comprehensive learning information including meaning, usage context, synonyms, antonyms, etymology, and examples across registers.

**Rules**:

**Coverage:**
- Read entire source thoroughly—identify all verbs that are decision-critical for understanding the text
- Prioritize: academic verbs, professional terminology, phrasal verbs, and verbs with nuanced meanings
- Extract comprehensively to reflect source richness; apply 80/20 principle (focus on high-frequency and high-impact verbs)
- Do not skip sections; cover both explicit verbs and verb forms that carry semantic weight

**Fidelity:**
- Ground all information in verbs actually present in source
- Never invent verbs not in source
- Derive meaning from context when source provides it; supplement with standard dictionary definitions

**Focus:**
- Action verbs with specific, precise meanings
- Phrasal verbs (verb + preposition/adverb combinations)
- Verbs with multiple meanings or registers
- Technical or domain-specific verbs
- Verbs that are challenging for language learners

**Format:**
- **Verb**: [infinitive form] (include verb + particle for phrasal verbs)
- **Meaning**: Collins Dictionary style definition(s)
  - *If verb has multiple senses, number them clearly*
  - *Use simple, clear language to explain meaning*
  - *Include grammatical patterns (transitive/intransitive)*
- **Synonyms**: 3-5 words/phrases with similar meaning
- **Antonyms**: 2-4 words/phrases with opposite meaning (if applicable)
- **When to Use**: Context, situations, or registers where this verb is appropriate
- **When NOT to Use**: Contexts where this verb would be inappropriate or incorrect
- **Common Phrases**: 3-5 collocations or fixed expressions using this verb
- **Root Analysis**: Break down word components (prefix + root + suffix)
- **Etymology**: Language origin and historical development (brief)
- **Examples [Casual]**: 2 natural example sentences in informal register
- **Examples [Formal]**: 2 natural example sentences in formal/academic register

**Output Format**:

Markdown ordered list only:
1. **Verb**: [infinitive]
   **Meaning**: 
   - [Collins-style definition with grammatical info]
   - *(Sense 2)* [Additional meaning if applicable]
   
   **Synonyms**: [word1], [word2], [word3]...
   
   **Antonyms**: [word1], [word2]...
   
   **When to Use**: [Context description - professional, academic, everyday situations]
   
   **When NOT to Use**: [Inappropriate contexts - too formal/informal, technical limitations]
   
   **Common Phrases**: 
   - [phrase 1]
   - [phrase 2]
   - [phrase 3]
   
   **Root Analysis**: [prefix] + [root] + [suffix] (meanings)
   
   **Etymology**: [Origin language] → [historical development] (brief)
   
   **Examples [Casual]**: 
   - [Natural conversational example 1]
   - [Natural conversational example 2]
   
   **Examples [Formal]**: 
   - [Academic/professional example 1]
   - [Academic/professional example 2]

**Instructions**:
- Use `1.` for every item; Markdown auto-numbers
- Output list only—no headings, comments, or explanations
- Focus on verbs that are essential for understanding source content about **Wallet**
- Prioritize verbs with learning value (nuanced, technical, or commonly misused)
