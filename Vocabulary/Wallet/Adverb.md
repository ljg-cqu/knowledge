# Extract Adverbs for Vocabulary Learning

**Task**: Extract adverbs (manner, degree, frequency, time, place) that are essential for language mastery from the source text about **Wallet**.

**Definition**: Each entry focuses on one adverb with comprehensive learning information including meaning, usage context, synonyms, antonyms, etymology, and examples across registers.

**Rules**:

**Coverage:**
- Read entire source thoroughly—identify all adverbs that are decision-critical for understanding the text
- Prioritize: adverbs of manner, degree modifiers, discourse markers, sentence adverbs, and adverbs with nuanced meanings
- Extract comprehensively to reflect source richness; apply 80/20 principle (focus on high-impact adverbs)
- Do not skip sections; cover both common and specialized adverbial expressions

**Fidelity:**
- Ground all information in adverbs actually present in source
- Never invent adverbs not in source
- Derive meaning from context when source provides it; supplement with standard dictionary definitions

**Focus:**
- Adverbs of manner (how something happens)
- Adverbs of degree (intensity, extent)
- Sentence adverbs (modify entire sentence, express attitude)
- Linking adverbs (discourse markers, transitions)
- Adverbs with precise or nuanced meanings

**Format:**
- **Adverb**: [base form] (indicate type: manner/degree/frequency/time/sentence)
- **Meaning**: Collins Dictionary style definition(s)
  - *If adverb has multiple senses, number them clearly*
  - *Use simple, clear language to explain meaning*
  - *Include information about position (front/mid/end position)*
- **Synonyms**: 3-5 words/phrases with similar meaning
- **Antonyms**: 2-4 words/phrases with opposite meaning (if applicable)
- **When to Use**: Context, situations, or registers where this adverb is appropriate
- **When NOT to Use**: Contexts where this adverb would be inappropriate or incorrect
- **Common Phrases**: 3-5 collocations or fixed expressions using this adverb
- **Root Analysis**: Break down word components (prefix + root + suffix)
- **Etymology**: Language origin and historical development (brief)
- **Examples [Casual]**: 2 natural example sentences in informal register
- **Examples [Formal]**: 2 natural example sentences in formal/academic register

Markdown ordered list only:
