# Token - Multi-Word Expressions

1. **Expression**: utility token (type: noun + noun collocation)
   **Meaning**: 
   - A digital token that provides access to specific products, services, or functions within a blockchain ecosystem, valued primarily for its functional use rather than investment returns
   - *Fixedness*: strong (established term with specific meaning in blockchain)
   
   **Synonyms**: functional token, access token, service token, usage token
   
   **Antonyms**: security token, investment token, equity token, governance-only token
   
   **When to Use**: When describing tokens that grant platform access, service payment rights, or network functionality usage; distinguishing from security tokens in regulatory contexts
   
   **When NOT to Use**: When describing tokens that represent ownership stakes or investment contracts with profit expectations primarily from others' efforts
   
   **Common Variations**: 
   - pure utility token
   - utility-based token
   - utility token model
   
   **Why This Partnership**: 
   - *For collocations*: "Utility" specifies the token's functional purpose, distinguishing it from other token types; this pairing became standard in blockchain terminology to clarify regulatory classification
   
   **Root Analysis**: Latin "utilitas" (usefulness) + Old English "tacen" (sign, symbol)
   
   **Examples [Casual]**: 
   - It's a utility token—you use it to pay fees on their network, not as an investment.
   - Utility tokens give you access to platform features instead of ownership rights.
   
   **Examples [Formal]**: 
   - The platform employs utility tokens to facilitate decentralized service access and incentivize network participation.
   - Utility tokens must demonstrate clear functional value propositions to avoid securities classification under regulatory frameworks.

1. **Expression**: governance token (type: noun + noun collocation)
   **Meaning**: 
   - A token that grants holders voting rights and decision-making power over protocol changes, parameter adjustments, treasury management, or development priorities
   - *Fixedness*: strong (standard terminology in blockchain governance)
   
   **Synonyms**: voting token, DAO token, decision-making token, protocol governance token
   
   **Antonyms**: non-voting token, passive token, utility-only token
   
   **When to Use**: When describing tokens enabling decentralized governance participation, DAO voting mechanisms, or protocol decision-making rights
   
   **When NOT to Use**: When tokens provide only utility or value transfer without voting or administrative functions
   
   **Common Variations**: 
   - governance and utility token (dual purpose)
   - pure governance token
   - governance voting token
   
   **Why This Partnership**: 
   - *For collocations*: "Governance" specifies the token's administrative function; this collocation emerged with DAOs (2016+) to distinguish voting rights tokens from other types
   
   **Root Analysis**: Greek "kybernan" (to steer) → governance + Old English "tacen" → token
   
   **Examples [Casual]**: 
   - Governance tokens let you vote on which features get added to the protocol.
   - If you hold enough governance tokens, you can submit proposals for changes.
   
   **Examples [Formal]**: 
   - Governance tokens distribute decision-making authority among stakeholders through on-chain voting mechanisms.
   - The protocol issues governance tokens to enable community-driven evolution while maintaining decentralization principles.

1. **Expression**: token economics (type: noun + noun collocation)
   **Meaning**: 
   - The economic model governing token supply, distribution, utility, and value accrual mechanisms; the study of token-based incentive structures
   - *Fixedness*: strong (often contracted to "tokenomics")
   
   **Synonyms**: tokenomics, token model, economic design, incentive structure, token system
   
   **Antonyms**: N/A (specialized technical term)
   
   **When to Use**: When analyzing or designing token supply mechanisms, distribution models, incentive structures, or discussing comprehensive economic frameworks
   
   **When NOT to Use**: When referring to general economics, traditional monetary policy, or non-blockchain financial systems
   
   **Common Variations**: 
   - tokenomics (portmanteau - most common)
   - token economic model
   - token economic design
   
   **Why This Partnership**: 
   - *For collocations*: "Token" + "economics" naturally combines to specify blockchain-native economic systems; frequently merged into "tokenomics" for brevity
   
   **Root Analysis**: Old English "tacen" + Greek "oikonomia" (household management)
   
   **Examples [Casual]**: 
   - The token economics are solid—limited supply with actual utility and burning mechanisms.
   - Bad token economics will kill a project faster than anything else.
   
   **Examples [Formal]**: 
   - Sound token economics require careful calibration of emission schedules, distribution mechanisms, and utility value propositions.
   - The protocol's token economics implement deflationary pressure through transaction-based burning paired with capped maximum supply.

1. **Expression**: burn mechanism (type: verb + noun collocation)
   **Meaning**: 
   - A protocol feature that permanently removes tokens from circulation by sending them to inaccessible addresses, reducing total supply
   - *Fixedness*: strong (standard terminology for deflationary features)
   
   **Synonyms**: burning protocol, destruction mechanism, deflationary mechanism, supply reduction system
   
   **Antonyms**: minting mechanism, emission mechanism, inflationary mechanism
   
   **When to Use**: When describing deflationary tokenomics features, fee destruction models, or supply reduction strategies
   
   **When NOT to Use**: When describing token creation, distribution, or mechanisms that don't reduce supply
   
   **Common Variations**: 
   - burning mechanism
   - token burn mechanism
   - automatic burn mechanism
   - burn-and-mint mechanism (if paired)
   
   **Why This Partnership**: 
   - *For collocations*: "Burn" metaphorically describes destruction, paired with "mechanism" to indicate systematic implementation; became standard in crypto for supply reduction features
   
   **Root Analysis**: Old English "byrnan" (to be on fire) + Greek "mēkhanē" (device)
   
   **Examples [Casual]**: 
   - Their burn mechanism destroys 2% of every transaction, making it deflationary over time.
   - The burn mechanism kicked in after EIP-1559, removing ETH from circulation permanently.
   
   **Examples [Formal]**: 
   - The protocol's burn mechanism implements systematic supply reduction through transaction fee destruction, creating deflationary pressure proportional to network utilization.
   - Burn mechanisms balance inflationary token emissions, approaching equilibrium supply levels where minting equals burning.

1. **Expression**: vesting schedule (type: gerund + noun collocation)
   **Meaning**: 
   - A predetermined timeline controlling when allocated tokens become transferable or accessible, typically featuring cliff periods and gradual release
   - *Fixedness*: strong (standard terminology in finance and blockchain)
   
   **Synonyms**: unlock schedule, release timeline, vesting period, token release schedule
   
   **Antonyms**: immediate release, instant availability, no vesting
   
   **When to Use**: When describing time-locked token releases for team members, investors, advisors, or any stakeholder allocations requiring gradual availability
   
   **When NOT to Use**: When tokens are immediately transferable, or when describing locks without gradual release mechanisms
   
   **Common Variations**: 
   - vesting timeline
   - token vesting schedule
   - vesting period
   - cliff vesting schedule
   
   **Why This Partnership**: 
   - *For collocations*: "Vesting" (legal term for ownership transfer) naturally pairs with "schedule" to indicate time-based progression; adopted from traditional equity compensation
   
   **Root Analysis**: Latin "vestire" (to clothe) + Old French "cedule" (list, timetable)
   
   **Examples [Casual]**: 
   - Team tokens have a 4-year vesting schedule with a 1-year cliff period.
   - The vesting schedule releases 25% of tokens every 6 months for 2 years.
   
   **Examples [Formal]**: 
   - Advisor allocations follow a 24-month linear vesting schedule with a 6-month cliff to align incentives with long-term project success.
   - Smart contracts enforce vesting schedules through time-locked transfer restrictions, automating token release according to predetermined intervals.

1. **Expression**: liquidity pool (type: noun + noun collocation)
   **Meaning**: 
   - A smart contract holding reserves of two or more tokens enabling automated trading through algorithmic market-making mechanisms
   - *Fixedness*: strong (fundamental DeFi concept with specific technical meaning)
   
   **Synonyms**: trading pool, AMM pool, liquidity reserve, token pool
   
   **Antonyms**: order book, centralized liquidity, illiquid market
   
   **When to Use**: When discussing decentralized exchange mechanisms, automated market makers, providing liquidity for trading, or DeFi yield farming
   
   **When NOT to Use**: When referring to centralized exchange order books, traditional market makers, or non-automated liquidity provision
   
   **Common Variations**: 
   - LP (abbreviation)
   - token liquidity pool
   - AMM liquidity pool
   - liquidity mining pool
   
   **Why This Partnership**: 
   - *For collocations*: "Liquidity" describes the pool's function (enabling trading), "pool" describes the structure (collective reserves); emerged with AMMs (2018+) as alternative to order books
   
   **Root Analysis**: Latin "liquidus" (fluid) + Old English "pol" (small body of water)
   
   **Examples [Casual]**: 
   - You can earn fees by adding your tokens to the liquidity pool on Uniswap.
   - Liquidity pools let you trade instantly without waiting for buyers or sellers.
   
   **Examples [Formal]**: 
   - Liquidity pools employ constant product formulas to determine token exchange rates algorithmically based on reserve ratios.
   - Impermanent loss affects liquidity pool participants when price ratios diverge significantly from initial deposit proportions.

1. **Expression**: market cap (type: noun + noun collocation - abbreviated)
   **Meaning**: 
   - The total value of a token calculated by multiplying circulating supply by current price, indicating project size and ranking
   - *Fixedness*: strong (standard financial metric adapted to crypto)
   
   **Synonyms**: market capitalization, token valuation, total value, market size
   
   **Antonyms**: individual token price, unit value
   
   **When to Use**: When comparing token project sizes, ranking cryptocurrencies, discussing valuation relative to competitors, or assessing investment scale
   
   **When NOT to Use**: When discussing individual token price rather than total value, or when fully diluted valuation is more relevant
   
   **Common Variations**: 
   - market capitalization (full form)
   - circulating market cap
   - fully diluted market cap
   - FDV (fully diluted valuation)
   
   **Why This Partnership**: 
   - *For collocations*: Shortened from "market capitalization," universally understood in finance; applied to crypto tokens to indicate total value
   
   **Root Analysis**: Latin "mercatus" (trade) + Latin "capitalis" (of the head, principal)
   
   **Examples [Casual]**: 
   - Bitcoin's market cap is over $500 billion—way bigger than any other crypto.
   - Don't just look at token price; check the market cap to see the real size.
   
   **Examples [Formal]**: 
   - Market capitalization provides relative valuation metrics enabling cross-project comparisons adjusted for supply differences.
   - Fully diluted market cap accounts for total maximum supply, providing comprehensive valuation including unvested or locked allocations.

1. **Expression**: gas fees (type: noun + noun collocation)
   **Meaning**: 
   - Transaction costs paid in native blockchain tokens to validators/miners for processing operations and executing smart contracts
   - *Fixedness*: strong (Ethereum-originated term, widely adopted across blockchains)
   
   **Synonyms**: transaction fees, network fees, processing costs, blockchain fees
   
   **Antonyms**: free transactions, zero-fee operations
   
   **When to Use**: When discussing transaction costs on Ethereum or EVM-compatible chains, explaining blockchain economics, or comparing network fees
   
   **When NOT to Use**: On non-EVM chains where different terminology applies (though increasingly adopted broadly), or when discussing fees unrelated to blockchain transactions
   
   **Common Variations**: 
   - gas costs
   - gas prices
   - transaction gas
   - gas fee estimation
   
   **Why This Partnership**: 
   - *For collocations*: "Gas" metaphorically represents computational fuel needed for transactions; "fees" indicates the cost; terminology originated from Ethereum's economic model
   
   **Root Analysis**: Dutch "gas" (chaos, empty space - coined by chemist) + Old French "fieu" (payment)
   
   **Examples [Casual]**: 
   - Gas fees are insane right now—it costs $50 just to send tokens on Ethereum.
   - Always check gas fees before transacting; sometimes waiting saves you money.
   
   **Examples [Formal]**: 
   - Gas fees vary dynamically based on network congestion, with users competing through fee bidding for transaction inclusion priority.
   - Layer-2 solutions substantially reduce gas fees by batching multiple transactions before settling on the main Ethereum chain.

1. **Expression**: smart contract (type: adjective + noun collocation)
   **Meaning**: 
   - Self-executing code deployed on blockchains that automatically enforces agreements and executes token-related functions when conditions are met
   - *Fixedness*: strong (fundamental blockchain concept with specific technical meaning)
   
   **Synonyms**: autonomous contract, blockchain contract, self-executing agreement, programmable contract
   
   **Antonyms**: traditional contract, manual agreement, paper contract
   
   **When to Use**: When discussing automated token mechanisms, DeFi protocols, NFT functionality, or any blockchain-based programmable logic
   
   **When NOT to Use**: When referring to traditional legal contracts without blockchain automation, or to describe blockchain nodes rather than code
   
   **Common Variations**: 
   - smart contracts (plural, referring to the technology generally)
   - contract code
   - deployed contract
   - smart contract protocol
   
   **Why This Partnership**: 
   - *For collocations*: "Smart" indicates autonomous, programmable nature; "contract" indicates agreement enforcement; coined by Nick Szabo (1990s), popularized by Ethereum
   
   **Root Analysis**: Old English "smeortan" (quick, clever) + Latin "contractus" (drawn together, agreement)
   
   **Examples [Casual]**: 
   - Smart contracts automatically handle the token swap without needing a middleman.
   - Once deployed, the smart contract can't be changed—the code is law.
   
   **Examples [Formal]**: 
   - Smart contracts enable trustless token transactions through cryptographically secured, immutably deployed programmable logic.
   - Security audits examine smart contract code for vulnerabilities before deployment to prevent exploitation and fund loss.

1. **Expression**: token standard (type: noun + noun collocation)
   **Meaning**: 
   - An established technical specification defining how tokens should be implemented to ensure compatibility and interoperability across wallets, exchanges, and applications
   - *Fixedness*: strong (technical terminology with specific meaning)
   
   **Synonyms**: token specification, token protocol, implementation standard, token interface
   
   **Antonyms**: custom implementation, non-standard token, proprietary format
   
   **When to Use**: When discussing token implementation specifications like ERC-20, ERC-721, BEP-20, or other technical standards ensuring ecosystem compatibility
   
   **When NOT to Use**: When referring to best practices rather than technical specifications, or when discussing native blockchain currencies that don't follow smart contract standards
   
   **Common Variations**: 
   - token specification
   - ERC token standard
   - standard compliance
   - token interface standard
   
   **Why This Partnership**: 
   - *For collocations*: "Token" specifies the subject; "standard" indicates established technical specification; this pairing emerged with ERC-20 (2015) to enable interoperability
   
   **Root Analysis**: Old English "tacen" + Old French "estandard" (rallying point, reference)
   
   **Examples [Casual]**: 
   - Most tokens follow the ERC-20 standard, so they work with any Ethereum wallet.
   - NFTs use the ERC-721 token standard to ensure marketplace compatibility.
   
   **Examples [Formal]**: 
   - Token standards define interface specifications enabling uniform implementation across decentralized applications and infrastructure providers.
   - Adherence to established token standards ensures cross-platform compatibility, facilitating ecosystem-wide adoption and integration.

1. **Expression**: staking rewards (type: gerund + noun collocation)
   **Meaning**: 
   - Tokens earned by participants who lock their holdings to support network security, liquidity provision, or protocol functions
   - *Fixedness*: strong (standard terminology in Proof-of-Stake and DeFi)
   
   **Synonyms**: staking yields, staking income, validator rewards, staking returns
   
   **Antonyms**: staking penalties, slashing, staking losses
   
   **When to Use**: When discussing incentives for Proof-of-Stake validation, liquidity provision returns, governance participation rewards, or any yield from token commitment
   
   **When NOT to Use**: When describing trading profits, airdrops, or income not earned through token staking/locking
   
   **Common Variations**: 
   - staking yield
   - staking APY/APR
   - validator rewards
   - staking incentives
   
   **Why This Partnership**: 
   - *For collocations*: "Staking" describes the action required; "rewards" indicates the compensation received; emerged with Proof-of-Stake systems (2012+)
   
   **Root Analysis**: Old English "staca" (stake, wager) + Old French "reward" (regard, consideration)
   
   **Examples [Casual]**: 
   - I'm earning 8% in staking rewards just by locking my tokens for 30 days.
   - Staking rewards compound over time if you restake them instead of cashing out.
   
   **Examples [Formal]**: 
   - Staking rewards incentivize network security through capital commitment, with compensation proportional to stake weight and duration.
   - Protocol staking rewards derive from transaction fees, inflation emissions, or revenue-sharing mechanisms depending on tokenomics design.

1. **Expression**: price discovery (type: noun + noun collocation)
   **Meaning**: 
   - The process through which token markets determine fair value through buyer-seller interactions, trading activity, and information dissemination
   - *Fixedness*: strong (established financial markets term applied to crypto)
   
   **Synonyms**: valuation process, market pricing, value determination, price finding
   
   **Antonyms**: price manipulation, artificial pricing, fixed price
   
   **When to Use**: When discussing how token markets establish value, analyzing liquidity's role in efficient pricing, or explaining market mechanisms
   
   **When NOT to Use**: When referring to discovering existing prices rather than the market formation process, or when price is administratively set
   
   **Common Variations**: 
   - efficient price discovery
   - price discovery mechanism
   - market price discovery
   - price discovery process
   
   **Why This Partnership**: 
   - *For collocations*: "Price" indicates what's being determined; "discovery" emphasizes the exploratory, market-driven process; from traditional financial markets terminology
   
   **Root Analysis**: Latin "pretium" (value, price) + Latin "discooperire" (uncover, reveal)
   
   **Examples [Casual]**: 
   - Low liquidity makes price discovery messy—the market doesn't know what the token should be worth.
   - Price discovery happens when buyers and sellers meet and figure out fair value through trading.
   
   **Examples [Formal]**: 
   - Efficient price discovery requires sufficient liquidity depth, information transparency, and rational market participant behavior.
   - Fragmented liquidity across multiple venues impairs price discovery efficiency, creating arbitrage opportunities and increasing transaction costs.

1. **Expression**: total value locked (type: adjective + noun + verb/adjective collocation)
   **Meaning**: 
   - The aggregate value of all tokens deposited in a DeFi protocol, measured in USD equivalent, indicating protocol size and adoption
   - *Fixedness*: strong (DeFi-specific metric, universally abbreviated as TVL)
   
   **Synonyms**: TVL, protocol deposits, locked liquidity, deposited value
   
   **Antonyms**: withdrawn value, available liquidity, circulating supply
   
   **When to Use**: When measuring DeFi protocol size, comparing platform adoption, tracking capital flows between protocols, or assessing protocol health
   
   **When NOT to Use**: When discussing market capitalization, token price, or metrics unrelated to deposited protocol assets
   
   **Common Variations**: 
   - TVL (universal abbreviation)
   - total locked value
   - protocol TVL
   - chain TVL (aggregated)
   
   **Why This Partnership**: 
   - *For collocations*: Phrase emerged in DeFi (2018+) to quantify protocol usage; "total" indicates aggregation, "value" measures in USD, "locked" emphasizes commitment
   
   **Root Analysis**: Latin "totus" (whole) + Latin "valor" (worth) + Old English "loc" (fasten)
   
   **Examples [Casual]**: 
   - The protocol's TVL hit $2 billion—that's huge growth in just six months.
   - When TVL drops suddenly, it usually means people are pulling funds out fast.
   
   **Examples [Formal]**: 
   - Total value locked serves as a proxy metric for protocol adoption, though it may overstate actual usage through circular lending relationships.
   - TVL migration patterns indicate competitive dynamics as liquidity responds to relative yield opportunities across protocols.

1. **Expression**: rug pull (type: noun + verb collocation - idiomatic)
   **Meaning**: 
   - *Literal meaning*: Pulling a rug from under someone
   - *Idiomatic meaning*: A scam where developers abandon a project and steal investors' tokens or liquidity, leaving holders with worthless assets
   - *Fixedness*: strong (crypto-specific idiom with technical implications)
   
   **Synonyms**: exit scam, liquidity theft, project abandonment, developer fraud
   
   **Antonyms**: legitimate project, trustworthy team, locked liquidity
   
   **When to Use**: Warning about potential scams, discussing project security red flags, explaining DeFi risks, or describing actual fraud events
   
   **When NOT to Use**: For legitimate project failures due to market conditions or technical issues; when accuracy requires distinguishing between types of fraud
   
   **Common Variations**: 
   - soft rug pull (gradual drain)
   - hard rug pull (immediate theft)
   - rug (verb: "they rugged")
   - rugpull (single word variant)
   
   **Why This Partnership**: 
   - *For idioms*: Metaphor of pulling rug out from under someone perfectly captures sudden loss of foundation; emerged in DeFi (2020) to describe liquidity theft
   
   **Root Analysis**: Middle English "rugge" (coarse fabric) + Old English "pullian" (to pluck)
   
   **Examples [Casual]**: 
   - That project was a total rug pull—devs took all the liquidity and disappeared.
   - Always check if liquidity is locked before investing; it prevents rug pulls.
   
   **Examples [Formal]**: 
   - Rug pull schemes exploit unaudited smart contracts with backdoor functions enabling unauthorized liquidity withdrawal by developers.
   - Liquidity locking mechanisms and multi-signature treasury controls mitigate rug pull risks by preventing unilateral developer fund access.

1. **Expression**: yield farming (type: noun + gerund collocation)
   **Meaning**: 
   - The practice of moving tokens between DeFi protocols to maximize returns through staking, liquidity provision, or lending, often involving complex strategies
   - *Fixedness*: strong (DeFi-specific term, universally understood)
   
   **Synonyms**: liquidity mining, DeFi farming, yield optimization, reward harvesting
   
   **Antonyms**: holding, passive investment, buy-and-hold
   
   **When to Use**: When discussing active DeFi strategies, protocol incentives, maximizing returns through token movement, or DeFi participation strategies
   
   **When NOT to Use**: For simple staking or holding, traditional investment strategies, or when farming metaphor seems unclear
   
   **Common Variations**: 
   - yield farm (verb form)
   - farming rewards
   - farm yields
   - farming strategy
   
   **Why This Partnership**: 
   - *For collocations*: "Yield" describes returns earned; "farming" metaphorically represents cultivating/harvesting rewards; emerged in DeFi Summer 2020 with liquidity mining programs
   
   **Root Analysis**: Old English "gieldan" (to pay) + Old English "feorm" (food provision, cultivation)
   
   **Examples [Casual]**: 
   - I'm yield farming across three protocols, constantly moving my tokens to chase the best APY.
   - Yield farming can be profitable but gas fees and impermanent loss eat into returns.
   
   **Examples [Formal]**: 
   - Yield farming strategies optimize risk-adjusted returns through diversified protocol exposure and automated rebalancing mechanisms.
   - Excessive yield farming incentives may attract mercenary capital exhibiting high velocity and rapid protocol exit following reward exhaustion.

1. **Expression**: double spend (type: adjective/noun + verb collocation)
   **Meaning**: 
   - Spending the same tokens twice by exploiting blockchain vulnerabilities or attacking consensus mechanisms before transactions are confirmed
   - *Fixedness*: strong (fundamental blockchain security concept)
   
   **Synonyms**: duplicate spending, double-spending attack, replay attack (related)
   
   **Antonyms**: secure transaction, confirmed transaction, immutable spending
   
   **When to Use**: When discussing blockchain security, explaining confirmation importance, analyzing consensus vulnerabilities, or describing attack vectors
   
   **When NOT to Use**: When referring to legitimate transaction retries, or when confusion with spending double the amount is possible
   
   **Common Variations**: 
   - double-spending (hyphenated gerund)
   - double spending attack
   - double-spend problem
   - prevent double-spending
   
   **Why This Partnership**: 
   - *For collocations*: "Double" indicates duplication; "spend" describes the transaction; this phrase predates blockchain, but solving double-spending was Bitcoin's key innovation
   
   **Root Analysis**: Latin "duplus" (twofold) + Old English "spendan" (to expend)
   
   **Examples [Casual]**: 
   - Blockchain prevents double-spending by recording every transaction permanently.
   - You can't double-spend Bitcoin because the network validates that you haven't already spent those coins.
   
   **Examples [Formal]**: 
   - Consensus mechanisms prevent double-spending attacks by requiring majority validation before transaction finalization.
   - Blockchain's distributed ledger architecture solves the double-spend problem without requiring trusted third-party intermediaries.
