# Node Vocabulary: Prepositions

1. **Preposition**: to (type: simple)
   **Meaning**: 
   - *(Abstract)* Indicating the recipient, target, or destination of an action or data
   - Used to show direction of data flow, function application, or target endpoints
   
   **Synonyms**: toward (direction), into (transformation), for (purpose)
   
   **Antonyms**: from (source), away from, out of
   
   **When to Use**: When describing data transmission, function parameters, API endpoints, or destination targets.
   
   **When NOT to Use**: Don't confuse with infinitive "to". Avoid overusing in nested structures.
   
   **Common Patterns**: 
   - **Prepositional phrases**: to the server, to the database, to production
   - **Verb + prep**: send to, pipe to, connect to, listen to, respond to
   - **Adjective + prep**: accessible to, available to
   - **Noun + prep**: connection to, request to, response to
   
   **Root Analysis**: Old English "to"
   
   **Etymology**: Old English "to" (toward) → Direction and purpose preposition
   
   **Examples [Casual]**: 
   - Send the data to the API endpoint.
   - Pipe the stream to the file.
   
   **Examples [Formal]**: 
   - The request is forwarded to the appropriate microservice.
   - Data flows from the client to the server through middleware layers.

   **Examples [Formal]**:
