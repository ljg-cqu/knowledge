1. **Preposition**: via (type: simple)
   **Meaning**: 
   - *(Spatial)* Travelling through a place on the way to another destination; by way of a particular route or location
   - *(Abstract)* By means of; through the agency or medium of something or someone; using a particular method, system, or person
   
   **Synonyms**: through (for routes), by way of (for intermediate stops), by means of (for methods), using (for instruments)
   
   **Antonyms**: directly to (not via intermediate point), without passing through
   
   **When to Use**: Use to indicate an intermediate point in a journey (spatial) or to specify the medium, method, or channel through which something is done or transmitted (abstract). Common in travel contexts, communication descriptions, and formal instructions.
   
   **When NOT to Use**: Do not use for simple instrument use (prefer "with" or "using"); avoid in very informal speech where "through" might be more natural. Not typically used for time expressions.
   
   **Common Patterns**: 
   - **Prepositional phrases**: via email, via satellite, via London, via the internet
   - **Verb + prep**: travel via, send via, communicate via, access via, transmit via
   - **Adjective + prep**: accessible via, available via, reachable via
   - **Noun + prep**: route via, connection via, transmission via, delivery via
   
   **Root Analysis**: Latin "via" meaning "way, road, path"
   
   **Etymology**: Latin "via" (road, way, path) → directly borrowed into English in 18th century, originally used in postal and travel contexts to indicate intermediate stopping points
   
   **Examples [Casual]**: 
   - I'm flying to Sydney via Singapore, so I'll have a few hours to explore the airport.
   - Just send me the document via email when you're done.
   
   **Examples [Formal]**: 
   - The data is transmitted via secure encrypted channels to ensure confidentiality.
   - Participants may register for the conference via the official website or by postal application.

1. **Preposition**: via satellite (type: compound phrasal)
   **Meaning**: 
   - *(Abstract)* Using satellite technology as the transmission medium; through satellite communication systems
   
   **Synonyms**: by satellite, through satellite transmission, using satellite technology
   
   **Antonyms**: via cable, via terrestrial transmission, via fiber optic
   
   **When to Use**: Use in technical, telecommunications, broadcasting, or scientific contexts to specify satellite-based transmission as the method. Common in discussions of communication infrastructure and remote connectivity.
   
   **When NOT to Use**: Avoid when the transmission method is obvious or irrelevant. Not appropriate for metaphorical uses.
   
   **Common Patterns**: 
   - **Prepositional phrases**: broadcast via satellite, transmitted via satellite, received via satellite
   - **Verb + prep**: transmit via satellite, send via satellite, communicate via satellite
   - **Adjective + prep**: available via satellite, accessible via satellite
   - **Noun + prep**: signal via satellite, connection via satellite, transmission via satellite
   
   **Root Analysis**: Latin "via" (way) + Latin "satelles" (attendant, guard) → compound describing transmission path through orbital technology
   
   **Etymology**: Modern compound (20th century) combining Latin "via" with "satellite" (from Latin "satelles"), popularized with space-age telecommunications
   
   **Examples [Casual]**: 
   - We get our TV channels via satellite because cable isn't available out here.
   - The GPS coordinates are updated via satellite every few seconds.
   
   **Examples [Formal]**: 
   - Remote communities receive educational programming via satellite broadcast systems.
   - Weather data is continuously monitored and transmitted via satellite to forecasting centers worldwide.

   **Etymology**: Late 20th century compound combining Latin "via" with "email" (coined 1970s-1980s), became standard business terminology in 1990s
   
   **Examples [Casual]**: 
   - Can you send me those photos via email? My phone storage is full.
   - I'll let you know via email once I hear back from them.
   
   **Examples [Formal]**: 
   - All applications must be submitted via email to the admissions office by the deadline.

   **Etymology**: Modern compound combining Latin "via" with "post" (from Latin "posita," referring to postal stations), formalized with modern postal systems
   
   **Examples [Casual]**: 
   - I ordered the book online, and they said they'd send it via post within a week.
   - You can still apply via post if you don't have internet access.
   
   **Examples [Formal]**: 
   - Written objections may be submitted via post to the address specified in the notice.
