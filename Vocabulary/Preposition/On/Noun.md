# Extract Nouns for Vocabulary Learning: "On"

**Task**: Extract nouns (concrete, abstract, technical terms, compound nouns) that are essential for language mastery and commonly used with or in contexts involving the preposition "**on**" from the source text.

**Definition**: Each entry focuses on one noun with comprehensive learning information including meaning, usage context, synonyms, antonyms, etymology, and examples across registers, with special attention to how these nouns relate to "on" constructions.

**Rules**:

**Coverage:**
- Read entire source thoroughly—identify all nouns that are decision-critical for understanding the text, especially those appearing in "noun + on" or "on + noun" patterns
- Prioritize: nouns that commonly appear with "on" prepositions, nouns in fixed "on" phrases, and nouns with contextual significance in "on" relationships
- Extract comprehensively to reflect source richness; apply 80/20 principle (focus on high-impact and frequently-used nouns with "on")
- Do not skip sections; cover both common and specialized terminology related to "on" usage

**Fidelity:**
- Ground all information in nouns actually present in source
- Never invent nouns not in source
- Derive meaning from context when source provides it; supplement with standard dictionary definitions

**Focus:**
- Nouns commonly following "on" (impact on, effect on, emphasis on, focus on, reliance on, dependence on)
- Nouns commonly preceding "on" in fixed phrases (look on, view on, perspective on, opinion on, comment on)
- Temporal nouns with "on" (Monday, arrival, departure, completion, occasion)
- Abstract concepts frequently used with "on" (basis, condition, understanding, assumption, grounds)
- Technical and domain-specific nouns in "on" contexts
- Nouns in idiomatic "on" expressions
