## Glossary

- **BIP-39**: Bitcoin Improvement Proposal 39, standard for generating 12-24 word mnemonic seed phrases from entropy, used by most HD wallets.
- **Guardian**: Trusted contact (friend, family, or device) in social recovery wallets who collectively approve wallet recovery via threshold signatures.
- **HD Wallet (Hierarchical Deterministic)**: Wallet deriving multiple private keys from single seed phrase using BIP-32/39/44 standards, enabling backup of all keys with one mnemonic.
- **HSM (Hardware Security Module)**: Tamper-resistant physical device storing cryptographic keys, used in MPC and custodial systems for enhanced security.
- **MPC (Multi-Party Computation)**: Cryptographic technique splitting private key into encrypted shares distributed across multiple parties, requiring threshold cooperation (e.g., 2-of-3) to sign transactions without reconstructing full key.
- **Multi-Signature Wallet**: Wallet requiring M-of-N signatures from multiple private keys to authorize transactions (e.g., 2-of-3, 3-of-5), common for institutional custody.
- **Seed Phrase**: 12-24 word mnemonic phrase (BIP-39) used to deterministically generate and recover all private keys in HD wallet; loss = permanent fund loss.
- **Social Recovery Wallet**: Non-custodial wallet using guardian network for recovery; users designate trusted contacts who collectively approve wallet restoration via threshold signatures (e.g., 3-of-5 guardians).
- **Threshold Signature**: Cryptographic scheme where M-of-N parties must cooperate to produce valid signature, used in social recovery and MPC wallets.
