## Glossary

- **BIP39 (Bitcoin Improvement Proposal 39)**: Standard defining seed phrase generation using 12-24 word mnemonic from predefined wordlist, providing 128-256 bit security for wallet recovery
- **Firmware**: Software embedded in hardware wallet device controlling cryptographic operations, requiring periodic security updates to patch vulnerabilities
- **Hardware wallet**: Physical device (USB-like) storing cryptocurrency private keys offline, providing security against malware and remote hacking vs software wallets
- **Metal backup**: Seed phrase engraved or stamped on corrosion-resistant metal plate (stainless steel, titanium), protecting against fire, water, and physical degradation unlike paper
- **Multi-location storage**: Security practice of storing seed phrase backup copies in geographically separate locations (e.g., home safe, bank vault, trusted contact) to eliminate single point of failure
- **Secure element chip**: Tamper-resistant chip in hardware wallet (used by Ledger) providing physical security against device extraction attacks, alternative to fully open-source approach
- **Seed phrase**: 12-24 word mnemonic backup generated during hardware wallet setup, used to recover wallet private keys if device lost/damaged, must be stored securely offline
- **Shamir Secret Sharing**: Cryptographic scheme splitting seed into N shares requiring M shares to recover (e.g., 3-of-5), distributing recovery authority vs single seed phrase backup
