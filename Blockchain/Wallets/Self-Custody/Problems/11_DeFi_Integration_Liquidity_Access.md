## Glossary

- **AMM (Automated Market Maker)**: Algorithm enabling decentralized exchanges to provide liquidity via mathematical formulas (e.g., constant product x*y=k) rather than order books
- **DEX (Decentralized Exchange)**: Peer-to-peer cryptocurrency exchange operating via smart contracts without centralized intermediary, using AMM or order book mechanisms
- **DeFi (Decentralized Finance)**: Blockchain-based financial services (lending, trading, yield farming) operating via smart contracts without traditional intermediaries
- **Flash loan**: Uncollateralized loan borrowed and repaid within single blockchain transaction, enabling arbitrage and also exploit attack vector
- **Impermanent loss (IL)**: Value erosion experienced by liquidity providers when token price ratio changes vs holding tokens outside pool, ranges 15-30% for volatile pairs
- **Layer-2 (L2)**: Blockchain scaling solution processing transactions off Ethereum mainnet to reduce gas costs (from $5-$50/tx to $0.10-$2/tx) while inheriting mainnet security
- **Liquidity pool**: Smart contract holding token pairs enabling decentralized trading, liquidity providers deposit tokens earning trading fees but risk impermanent loss
- **Liquidity provider (LP)**: User depositing token pairs into DeFi pools, earning trading fees (0.3% per swap) and farming rewards but risking impermanent loss
- **TVL (Total Value Locked)**: Dollar value of assets deposited in DeFi protocol, key metric for protocol adoption and liquidity depth (e.g., Aave $10B TVL)
- **Yield farming**: DeFi strategy depositing assets into protocols to earn returns via trading fees, interest, and token rewards, requiring active management and risk assessment
