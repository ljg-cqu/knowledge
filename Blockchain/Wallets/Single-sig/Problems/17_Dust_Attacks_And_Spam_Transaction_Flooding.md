Additionally, mempool spam transactions flood networks causing congestion, fee spikes, and confirmation delays for legitimate users [Web: Lightspark, 2024]. Combined impact: privacy erosion (user identity linking), financial losses ($83.8M from address poisoning alone), operational disruption (degraded UX from spam), and wallet interface clutter (hundreds of dust transactions obscuring real transactions).

## Glossary

- **Address poisoning**: Attack exploiting user copy-paste behavior where attacker generates lookalike cryptocurrency address matching first/last characters of victim's address, sends minimal transaction populating victim's history, causing victim to copy attacker's address for subsequent transactions; caused $83.8M losses across 6,633 incidents.
- **Address whitelisting**: Security feature allowing users to pre-approve list of trusted recipient addresses; non-whitelisted addresses require additional verification steps; reduces address poisoning risk 80-90% when adopted but <10% user adoption due to UX friction.
- **BRC-20**: Token standard on Bitcoin blockchain using Ordinals protocol; caused major mempool spam in 2023 with weeks of congestion and 10-20× fee spikes pricing out legitimate transactions.
- **Dust attack**: Privacy attack sending tiny cryptocurrency amounts (5-10 satoshis Bitcoin, 0.000001 ETH Ethereum) to thousands of addresses to enable transaction graph analysis linking addresses for de-anonymization; targets 5-10% of wallet addresses historically.
- **Dust limit**: Minimum economically viable transaction output; Bitcoin 5-10 satoshis; transactions below dust limit uneconomical to spend due to fee exceeding value; attackers intentionally send sub-dust amounts to clutter wallets and enable tracking.
- **ENS (Ethereum Name Service)**: Decentralized naming system mapping human-readable names (alice.eth) to Ethereum addresses; reduces address copy-paste errors and poisoning risk; 2M+ registrations but <0.5% of addresses, limiting effectiveness.
- **Levenshtein distance**: Edit distance algorithm measuring similarity between two strings by counting minimum single-character edits (insertions, deletions, substitutions) needed to transform one into other; used for lookalike address detection.
- **Lookalike address**: Cryptocurrency address intentionally generated to visually resemble legitimate address by matching first/last characters; exploits human address verification limitations and copy-paste habits; enables address poisoning attacks causing $83.8M losses.
- **Mempool**: Memory pool of pending blockchain transactions awaiting validator/miner confirmation; spam attacks flood mempool with thousands of low-fee transactions causing congestion, confirmation delays (5-15min), and fee spikes (2-5×).
- **Mempool spam**: Network attack flooding transaction memory pool with thousands of low-fee transactions consuming capacity; delays legitimate transaction confirmation, forces fee overpayment, degrades user experience; affects all users during attack periods.
- **Ordinals**: Bitcoin protocol enabling inscription of arbitrary data on satoshis creating NFT-like tokens; BRC-20 token standard using Ordinals caused major mempool spam 2023 with weeks of congestion.
- **SNS (Solana Name Service)**: Decentralized naming system for Solana blockchain mapping human-readable names to addresses; <100K registrations limiting effectiveness for address poisoning prevention.
- **UTXO (Unspent Transaction Output)**: Bitcoin's accounting model where each transaction consumes previous outputs and creates new outputs; enables transaction graph analysis linking addresses when UTXOs consolidated; dust attacks exploit UTXO model to de-anonymize users through graph clustering.
- **UTXO management**: Advanced wallet feature allowing users to view, select, and freeze individual unspent transaction outputs; enables avoiding dust UTXOs to preserve privacy; <1% user adoption due to technical complexity.
- **Wrench attack**: Physical violence or coercion to steal cryptocurrency; targets high-value users identified through de-anonymization from dust attacks, blockchain analysis, or social media; growing concern with increased crypto adoption.
