## Glossary

- **ADA (Americans with Disabilities Act)**: US federal law requiring accessible digital services, enforced via Section 508 standards
- **Alt text**: Alternative text description for images/visual content, enabling screen reader comprehension
- **Assistive technology**: Hardware/software enabling disabled users to interact with digital systems (screen readers, voice control, keyboard navigation)
- **Cognitive impairment**: Disability affecting memory, attention, comprehension, decision-making abilities
- **EAA (European Accessibility Act)**: EU regulation requiring digital accessibility compliance by June 2025
- **JAWS (Job Access With Speech)**: Popular Windows screen reader software for blind/visually impaired users
- **Motor impairment**: Disability affecting physical movement and control, limiting mouse/touchscreen use
- **NVDA (NonVisual Desktop Access)**: Free open-source Windows screen reader
- **Screen reader**: Software that reads on-screen text aloud, primary assistive technology for blind/visually impaired users
- **TalkBack**: Android's built-in screen reader
- **Touch target**: Interactive element size recommendation (minimum 44x44px for accessibility)
- **Visual impairment**: Disability affecting vision, ranging from low vision to complete blindness
- **VoiceOver**: Apple's built-in screen reader for iOS and macOS
- **WCAG (Web Content Accessibility Guidelines)**: International standard for web accessibility, levels A/AA/AAA compliance
- **WCAG 2.1 AA**: Industry-standard accessibility compliance level, legally required in many jurisdictions
