## Glossary

- **6ix1een attack**: Key extraction attack on GG18/GG20 threshold signature protocols requiring only 16 signatures to recover private key via Paillier modulus factorization
- **CGGMP21**: Third-generation threshold signature protocol (Canetti-Gennaro-Goldfeder-Makriyannis-Peled, 2021) with UC-security guarantees and 1-round online signing
- **CVE-2023-33241 (BitForge)**: Vulnerability in CCLST implementation enabling key extraction through signature malleability
- **Death by 1M cuts**: Key extraction attack on GG18/GG20 exploiting weak beta parameter, requires ~1 million signatures but practical for long-running systems
- **DKLS23**: Threshold signature protocol optimized for 2-party scenarios (Doerner-Kondi-Lee-Shelat, 2023), deployed in Coinbase custody
- **GG18**: First-generation threshold ECDSA protocol (Gennaro-Goldfeder, 2018, 9-round) enabling distributed Bitcoin/Ethereum signing
- **GG20**: Second-generation threshold ECDSA protocol (Gennaro-Goldfeder, 2020, 7-round optimization) with identifiable abort and enhanced proofs
- **Key refresh**: Proactive security mechanism generating new key shares while maintaining same public key, enables security improvement without full migration
- **MPC-CMP**: Threshold signature protocol achieving single-round online signing with continuous key rotation for proactive security
- **Paillier encryption**: Homomorphic encryption enabling addition on ciphertexts, used in GG18/GG20 but vulnerable if modulus contains small primes
- **Threshold signature**: Cryptographic protocol requiring M-of-N parties to cooperate for signing, distributing trust and eliminating single point of failure
- **TSSHOCK**: Vulnerability disclosure for key extraction attacks on threshold signature schemes including CGGMP21, requiring protocol updates
- **UC-security (Universal Composability)**: Formal security framework guaranteeing security properties preserved when protocols composed with other protocols
- **Zero-downtime migration**: Upgrading production systems without service interruption, critical for 24/7 institutional trading operations
