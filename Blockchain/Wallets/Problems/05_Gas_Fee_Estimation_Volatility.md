## Glossary

- **Base fee**: Minimum fee per gas unit required for transaction inclusion, algorithmically adjusted per block based on congestion, burned (not paid to validators)
- **EIP-1559**: Ethereum Improvement Proposal introducing base fee + priority tip model replacing pure auction, implemented August 2021
- **EIP-4844 (Proto-danksharding)**: Ethereum upgrade introducing blob transactions for Layer-2 data posting, reducing L2 fees significantly, implemented March 2024 (Dencun upgrade)
- **Gas**: Computational unit measuring transaction execution cost on blockchain; total fee = gas used × gas price
- **Gas limit**: Maximum gas units user willing to spend for transaction; unused gas refunded
- **Gwei**: Gas price unit (1 gwei = 0.000000001 ETH); common denomination for fee display
- **Layer-2 (L2)**: Scaling solution processing transactions off main chain, posting compressed data to Layer-1 for security
- **Mempool**: Queue of pending transactions waiting for block inclusion
- **MEV (Maximal Extractable Value)**: Profit extracted by reordering/inserting/censoring transactions in blocks
- **Priority fee (tip)**: Additional fee paid to validators for faster transaction inclusion, user-specified
- **Transaction simulation**: Pre-execution of transaction to estimate gas usage and preview state changes
