## Glossary

- **Crocodilus malware**: Android trojan using overlay attacks, remote access, and social engineering to drain cryptocurrency wallets, 300K+ infections 2024-2025
- **Overlay attack**: Malware technique displaying fake interface over legitimate app to capture user credentials or seed phrases
- **Pixnapping**: Android zero-click vulnerability reconstructing screen content pixel-by-pixel to capture sensitive information (seed phrases, 2FA codes) without user interaction
- **Secure Enclave**: Apple hardware security chip (iPhone 5S+) providing isolated key storage and cryptographic operations
- **SIM swap attack**: Social engineering attack convincing carrier to transfer victim's phone number to attacker-controlled SIM, bypassing SMS-based 2FA
- **TEE (Trusted Execution Environment)**: Hardware-isolated secure area in Android devices for sensitive operations (key storage, authentication)
- **Zero-click exploit**: Security vulnerability exploitable without any user interaction (no clicking, no opening, no approval required)
