## Glossary

- **Blockchain indexer**: Service that processes and indexes blockchain data into queryable databases, providing APIs for efficient data retrieval vs. direct RPC queries (e.g., The Graph, Covalent, Moralis).
- **Blockchain reorganization (reorg)**: Event where blockchain replaces blocks in its canonical chain with alternative blocks, typically during network forks or 51% attacks, causing temporary balance fluctuations and transaction reversals.
- **ERC-20**: Ethereum token standard defining common interface for fungible tokens on Ethereum and EVM-compatible chains (Polygon, BSC, Avalanche, Arbitrum, Optimism).
- **Price oracle**: Service providing off-chain data (like token prices) to blockchain smart contracts or applications, aggregating data from exchanges and trading venues (e.g., Chainlink, Band Protocol).
- **RPC (Remote Procedure Call)**: Protocol enabling applications to interact with blockchain nodes, used to query balances, transactions, smart contract states, and submit transactions (e.g., Ethereum JSON-RPC).
- **RPC endpoint**: URL providing access to blockchain node RPC interface, typically rate-limited and may charge per request (e.g., Infura, Alchemy, QuickNode endpoints).
- **SPL token**: Solana Program Library token standard for fungible and non-fungible tokens on Solana blockchain.
- **Token standard**: Protocol defining how tokens are implemented on blockchain, enabling interoperability and consistent wallet/exchange integration (e.g., ERC-20, BEP-20, SPL).
- **UTXO (Unspent Transaction Output)**: Bitcoin's transaction model where funds exist as discrete outputs that can be spent, requiring different balance calculation vs. Ethereum's account model.
- **Websocket**: Communication protocol enabling persistent, bidirectional connection between client and server for real-time data streaming, used for live blockchain event monitoring.
