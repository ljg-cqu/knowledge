## Glossary

- **API gateway**: Middleware layer managing API requests, enforcing rate limits, authentication, routing, caching, and load balancing.
- **API rate limit**: Maximum number of requests per time period (e.g., 1000 requests/min) to prevent resource exhaustion.
- **Arbitrage**: Strategy exploiting price differences for same asset across different exchanges to generate risk-free profit.
- **CEX (Centralized Exchange)**: Cryptocurrency exchange operated by central entity (e.g., Binance, Coinbase, Kraken) with order books and custody.
- **Circuit breaker**: Automated mechanism stopping operations when thresholds exceeded (e.g., error rates, latency) to prevent cascading failures.
- **DEX (Decentralized Exchange)**: Peer-to-peer cryptocurrency exchange without intermediary, using smart contracts (e.g., Uniswap, SushiSwap).
- **EMS (Execution Management System)**: Software enabling trade execution, order routing, and real-time market data access.
- **Follow-the-sun**: Operations model with geographically distributed teams providing continuous 24/7 coverage across time zones.
- **Gas fee**: Transaction cost on Ethereum and EVM-compatible blockchains paid to validators, variable based on network congestion.
- **Horizontal scaling**: Adding more machines/nodes to system to increase capacity, versus vertical scaling (upgrading single machine).
- **HSM (Hardware Security Module)**: Tamper-resistant hardware device securing private keys with cryptographic operations.
- **L2 (Layer 2)**: Scaling solution built atop blockchain (e.g., Arbitrum, Optimism on Ethereum) offering faster, cheaper transactions.
- **MPC (Multi-Party Computation)**: Cryptographic technique distributing private key shares across parties for secure distributed signing.
- **OTC (Over-The-Counter)**: Trading directly between parties without exchange order book, typical for large institutional transactions.
- **Perpetual contract**: Derivative without expiry date tracking underlying asset price, popular for leveraged trading.
- **PMS (Portfolio Management System)**: Software for portfolio tracking, performance analysis, risk management, and reporting.
- **Proof-of-reserves**: Cryptographic verification demonstrating custodian controls private keys for claimed holdings, often audited.
- **Slippage**: Difference between expected trade price and actual execution price, higher during low liquidity or large orders.
- **Throughput**: Number of transactions system can process per unit time (e.g., 10K transactions/day).
- **TradFi (Traditional Finance)**: Conventional financial system (banks, stock exchanges, brokers) versus DeFi (decentralized finance).
