## Glossary

- **Air-gapped**: Physically isolated from internet and network connections, preventing remote attacks but requiring manual processes for any data transfer.
- **Cold storage/wallet**: Offline storage method for private keys using hardware devices, paper backups, or USB drives disconnected from internet for maximum security.
- **Defense-in-depth**: Security strategy using multiple layers of controls (firewalls, intrusion detection, access controls, monitoring) so failure of one layer doesn't compromise entire system.
- **Hot wallet**: Internet-connected wallet enabling immediate transaction signing for trading and DeFi operations, but exposed to remote attack vectors.
- **HSM (Hardware Security Module)**: Dedicated cryptographic hardware device that securely generates, stores, and manages private keys with tamper-resistant properties and network connectivity.
- **MPC (Multi-Party Computation)**: Cryptographic technique distributing key shares across multiple parties enabling transaction signing without reconstructing full key, eliminating single point of failure.
- **Multi-signature (Multi-sig)**: Wallet requiring multiple private keys (e.g., 2-of-3, 3-of-5) to authorize transactions, preventing single-key compromise.
- **Threshold signature**: Cryptographic scheme where subset (t-of-n) of parties can collectively generate valid signature without reconstructing private key.
- **Warm wallet**: Intermediate security tier with enhanced controls compared to hot wallets but more accessible than cold storage, typically requiring manual approval but maintaining network connectivity.
