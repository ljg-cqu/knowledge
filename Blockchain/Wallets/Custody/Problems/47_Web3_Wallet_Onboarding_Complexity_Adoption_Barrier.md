## Problem Statement

## Glossary

- **Account Abstraction (ERC-4337)**: Ethereum standard enabling smart contract wallets with flexible authentication (social recovery, biometrics, session keys), deployed March 2023
- **BIP-39**: Bitcoin Improvement Proposal 39 defining standard for mnemonic seed phrases (12-24 words)
- **Gas Fees**: Transaction fees on blockchain networks, vary 10-100x based on congestion (Ethereum 20-200+ gwei)
- **Guardian**: Trusted contact in social recovery system who can collectively authorize account recovery
- **Hardware Wallet**: Physical device storing private keys offline (Ledger, Trezor), costs $50-500
- **MPC (Multi-Party Computation)**: Cryptographic protocol distributing key shares, used by seedless wallets (ZenGo)
- **Seed Phrase**: 12-24 word mnemonic for wallet recovery, loss means permanent asset loss
- **Social Recovery**: Wallet recovery mechanism using trusted guardians, adoption <5%
- **Testnet**: Separate blockchain network for testing with valueless tokens
- **Time-to-Confident-Usage**: Duration from wallet installation to user comfort with transactions (currently 2-7 days)
