# Extract Phrases & Idioms for Vocabulary Learning

**Task**: Extract phrases, idioms, collocations, and multi-word expressions that are essential for language mastery from the source text.

**Definition**: Each entry focuses on one phrase or idiomatic expression with comprehensive learning information including meaning, usage context, synonyms, etymology, and examples across registers.

**Rules**:

**Coverage:**
- Read entire source thoroughly—identify all phrases and idioms that carry special meaning beyond literal interpretation
- Prioritize: idiomatic expressions, fixed phrases, collocations, phrasal constructions, and domain-specific multi-word terms
- Extract comprehensively to reflect source richness; apply 80/20 principle (focus on high-frequency and culturally significant phrases)
- Do not skip sections; cover both transparent and opaque multi-word expressions

**Fidelity:**
- Ground all information in phrases actually present in source
- Never invent phrases not in source
- Derive meaning from context when source provides it; supplement with standard dictionary definitions

**Focus:**
- Idiomatic expressions (non-literal meaning)
- Fixed phrases and binomials (e.g., "salt and pepper", "pros and cons")
- Collocations (natural word pairings)
- Phrasal constructions with specialized meanings
- Domain-specific multi-word terms
- Conversational formulae and discourse markers

**Format:**
- **Phrase/Idiom**: [complete multi-word expression]
- **Meaning**: Collins Dictionary style definition(s)
  - *Explain both literal and figurative/idiomatic meaning if applicable*
  - *Use simple, clear language to explain meaning*
  - *Include register information (formal/informal/neutral)*
- **Synonyms**: 3-5 equivalent phrases or single-word alternatives
- **Antonyms**: 2-4 phrases with opposite meaning (if applicable)
- **When to Use**: Context, situations, or registers where this phrase is appropriate
- **When NOT to Use**: Contexts where this phrase would be inappropriate, sound unnatural, or be misunderstood
- **Common Variations**: Alternative forms or related phrases (if applicable)
- **Root Analysis**: Etymology of key components (for non-idiomatic phrases)
- **Etymology**: Origin story, historical/cultural background of the phrase
- **Examples [Casual]**: 2 natural example sentences in informal register
- **Examples [Formal]**: 2 natural example sentences in formal/academic register (or note if phrase is register-restricted)

**Output Format**:

Markdown ordered list only:
1. **Phrase/Idiom**: [complete expression]
   **Meaning**: 
   - [Collins-style definition]
   - *Literal*: [if different from figurative]
   - *Figurative/Idiomatic*: [actual intended meaning]
   
   **Synonyms**: [phrase1], [phrase2], [single word alternative]...
   
   **Antonyms**: [phrase1], [phrase2]...
   
   **When to Use**: [Context description - cultural situations, emotional tone, register]
   
   **When NOT to Use**: [Inappropriate contexts - formal writing if too casual, cultural misunderstandings]
   
   **Common Variations**: 
   - [variation 1]
   - [variation 2]
   
   **Root Analysis**: [key components] (if relevant for non-idiomatic phrases)
   
   **Etymology**: [Origin story - historical/cultural background, first known use] (brief)
   
   **Examples [Casual]**: 
   - [Natural conversational example 1]
   - [Natural conversational example 2]
   
   **Examples [Formal]**: 
   - [Academic/professional example 1] (or *"Typically informal; avoid in formal contexts"*)
   - [Academic/professional example 2]

**Instructions**:
- Use `1.` for every item; Markdown auto-numbers
- Output list only—no headings, comments, or explanations
- Focus on phrases that are essential for natural, fluent communication
- Prioritize phrases with learning value (idiomatic, culturally significant, or commonly misused)
- For phrases that are register-specific, clearly note restrictions
