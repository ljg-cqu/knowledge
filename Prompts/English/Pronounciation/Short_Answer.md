# Short Answer: English Pronunciation

Generate 30–50 pronunciation problems with IPA transcriptions, accent comparisons, phonetic analysis, and citations.

Use this when you need exam-style, decision-critical pronunciation practice for intermediate–advanced adult learners of English phonetics/phonology, where accuracy and accent coverage materially affect assessment or learning decisions.

## Requirements

- **Context**: Intermediate–advanced adult learners; English pronunciation/phonetics short-answer problems.
- **Scale & timeline**: 30–50 problems per set, usable as a 1–2 week unit in a course or exam prep.
- **Constraints**: Accent-balanced, citation-backed, IPA-heavy; answers must stay within short-answer scope (no long essays).

| Parameter | Specification |
|-----------|---------------|
| **Scope** | 30–50 problems; difficulty 20/40/40 (foundational/intermediate/advanced) |
| **Types** | IPA transcription, minimal pairs, stress, accent comparison, connected speech, phoneme ID, suprasegmentals |
| **Focus** | Non-trivial, decision-relevant contrasts (meaning/comprehension/exam-impact); avoid purely mechanical or trivial dictionary lookups |
| **Answers** | IPA + audio description + 2–4 step phonetic analysis; allow accent variation |
| **Sources** | ~60% EN, ~30% ZH, ~10% other; APA 7th; textbooks + peer-reviewed + dictionaries |
| **Grading** | Accept accent variants; partial credit: 70% (phonemes), 50% (placement) |
| **Quality** | Real-world scenarios; state dialect; flag regional variants; include alternatives |

## Reference Floors

| Section | Min | Requirements |
|---------|-----|--------------|
| Glossary | ≥15 | Phonetic terms, articulatory features; `**G#: Term**: Definition [Lang]` |
| Resources | ≥8 | Dictionaries, IPA charts, audio databases; updated ≤24mo, authoritative |
| Literature | ≥8 | Textbooks, peer-reviewed; phonological rules, acoustic analysis |
| APA | ≥15 | Language ~60/30/10; DOIs/archived URLs; accent studies |

**Citation**: `[Ref: ID]` (G#/R#/L#/A#). **Exception**: State shortfall + rationale.

## Quality Gates

| Gate | Pass Criteria |
|------|---------------|
| Recency | ≥60% last 5yr |
| Diversity | ≥4 accents (GA, RP, Indian, Australian); ≥5 phonetic categories; no source >20% |
| Coverage | ≥75% cite ≥1; ≥40% cite ≥2; ≥80% IPA |
| Links | All resolve or archived |
| Cross-Refs | All `[Ref: ID]` resolve |
| Accent Balance | ≥40% GA, ≥30% RP, ≥20% other |

**Scaling**: >50 problems → floors ×1.5

## Workflow

### 1. Plan Topics
- 5–7 clusters: Vowels, Consonants, Suprasegmentals, Connected Speech, Accent Comparison, Problematic Sounds, Phonological Rules
- 5–9 problems/cluster → 30–50 total; difficulty 20/40/40; balance: ≥40% GA, ≥30% RP, ≥20% other
- Prefer high-impact patterns (meaning-changing minimal pairs, common learner confusions, assessment-critical contrasts); exclude pure history, abstract theory, and ultra-rare edge cases (<5% frequency) unless they materially affect grading or comprehension.
- **Verify**: Total 30–50, difficulty ≈20/40/40, accent distribution met

### 2. Collect References
- Gather ≥15 glossary, ≥8 resources, ≥8 literature, ≥15 APA
- Tag language ([EN]/[ZH]), year, accent, category; assign IDs (G#/R#/L#/A#)
- **Verify**: Floors met, language ≈60/30/10, recency ≥60% 5yr, ≥4 accents

### 3. Generate Problems
- State: target word/phrase, accent(s), difficulty, type, context, constraints
- Include: ≥1 `[Ref: ID]`, IPA, accent variants, 2–4 analysis steps, articulatory descriptions (place/manner/voicing; height/backness/roundness)
- **Verify every 5**: Completeness, IPA accuracy, variants, citations, descriptions

### 4. Document Grading
- Alternatives: accent variants, registers, connected speech forms
- Partial credit: 70% (phonemes), 50% (placement); L1 interference patterns
- **Verify**: All have rubrics + alternatives + learner errors

### 5. Compile References
- Populate: Glossary, Resources, Literature, APA
- Include: accent coverage, IPA version, audio, update frequency; verify `[Ref: ID]` resolve
- Check: APA 7th + language tags + DOIs/archived URLs + accent tags
- Do not invent references; if no authoritative source exists for a pattern, soften claims or tag it explicitly as `[Assumption]` with a brief rationale.

### 6. Validate (ALL 14 steps)

1. **Counts**: `G: X (≥15) | R: Y (≥8) | L: Z (≥8) | A: W (≥15) | Problems: N (F/I/A)` → Floors met, difficulty ≈20/40/40
2. **Coverage**: `X of Y cite ≥1 (Z%); W cite ≥2 (V%)` → ≥75% cite ≥1, ≥40% cite ≥2
3. **Language**: `EN: X (Y%) | ZH: A (B%) | Other: C (D%)` → EN ≈50-70%, ZH ≈20-40%, Other ≈5-15%
4. **Recency**: `X of Y (Z%) from 2020-2025` → ≥60% last 5yr
5. **Diversity**: `GA: X | RP: Y | Indian: Z | Australian: W | Other: V | Accents: N | Max: M (P%)` → ≥4 accents, no source >20%
6. **Accent Balance**: `GA: X (Y%) | RP: A (B%) | Other: C (D%)` → GA ≥40%, RP ≥30%, Other ≥20%
7. **IPA Coverage**: `X of Y include IPA (Z%)` → ≥80% IPA
8. **Links**: `Tested X: Y ok, Z broken` → All ok OR archived
9. **Cross-Refs**: `X refs: Y ok, Z broken` → Z=0
10. **Solutions**: `X of Y complete; Z incomplete` → Z=0
11. **Phonetic Categories**: `Vowels: X | Consonants: Y | Suprasegmentals: Z | Connected: W | Comparison: V` → ≥5 categories
12. **Articulatory Descriptions**: `X of Y have descriptions; Z missing` → Z=0
13. **Accent Variants**: `X of Y include variants; Z missing` → ≥80% variants
14. **Learner Errors**: `X of Y document errors; Z missing` → ≥70% L1 interference

During checks 1–14, explicitly self-review phonetic claims against cited sources, flag any assumptions or unresolved conflicts (e.g., dictionary vs corpus vs audio), and, where needed, mark them as `[Assumption]` in the problems instead of silently inventing rules or variants.

**MANDATORY**: If fail → fix → regenerate → re-validate. Proceed only when all pass.

### 7. Checklist

- [ ] Floors: G≥15, R≥8, L≥8, A≥15; Difficulty 20/40/40; Language ~60/30/10
- [ ] Recency ≥60% 5yr; Diversity ≥4 accents, no source >20%

- [ ] Coverage ≥75% cite ≥1, ≥40% cite ≥2; IPA ≥80%
- [ ] Quality: 2-4 steps, IPA accuracy, variants, articulatory descriptions, learner errors
- [ ] Resources: Accent coverage, IPA version, audio, update ≤24mo; links/refs resolve; alternatives

  - [Consonants](#consonants): [P5](#p5), [P6](#p6)
  - [Suprasegmentals](#suprasegmentals): [P10](#p10), [P11](#p11)

### Problem Template

```markdown
#### P#: [Title] - [Word/Phrase]
**Difficulty**: [F/I/A] | **Type**: [IPA/Minimal Pair/Stress/Accent/Connected/Phoneme ID]
**Accents**: [GA/RP/Indian/Australian/etc.]

**Problem**: [Statement with target + context; cite [Ref: ID]]
**Given**: [Orthographic + phonetic context + target accent(s)]
**Assumptions**: [Register, speech rate, dialect; cite [Ref: ID]]

**Answer**: [IPA per accent] | **Tolerance**: [Regional variants accepted]

**Solution**:
1. [Phonemic analysis [Ref: ID]]
2. [Articulatory: place/manner/voicing OR height/backness/roundness]
3. [Accent-specific variants]
4. [Connected speech effects]

**Regional Variants**: [By region/register]
**Acoustics**: [Formants, duration, pitch when relevant]
**L1 Interference**: [Expected errors by L1]
**Alternatives & Trade-offs**: [≥2 valid variants or teaching approaches with pros/cons and when each is appropriate; include at least one case where the pattern/solution should NOT be generalized]
**Grading**: [Rubric + partial credit] | **Common Errors**: [List + reasoning]
```

### Visual Elements

- **Tables**: Minimal pairs, vowel/consonant charts, stress patterns, accent comparisons
- **Lists**: Numbered (steps), bulleted (variants, features)
- **IPA**: Unicode: `/phonemic/`, `[phonetic]`
- **Diagrams**: Vowel quadrilaterals (Mermaid), articulation charts, waveforms
- **Audio**: Link to databases when available

### Reference Formats

**Glossary**: `**G#: Term**: Definition [Lang]` + IPA when relevant

**Resources**: `**R#: Name** ([lang])` + Type, Accents, IPA version, Audio, Last update

**Literature**: `**L#: Title** (Year) ([lang])` + Core, Methodology, Impact, Limitations, APA + link

**APA**: By language; APA 7th + tag + accent tag + link

**Example**:
```text
**G1: Schwa**: Mid-central vowel /ə/, most common in English, unstressed [EN]

**R1: Cambridge English Pronunciation Dictionary** ([EN])
Type: Online dictionary | Accents: GA, RP | IPA: 2015 | Audio: Native recordings | Update: 2024-06

**L1: English Phonetics and Phonology** (Roach, 2020) ([EN])
Core: Vowel reduction, stress-timing | Methodology: RP/GA corpus | Impact: 15K+ citations | Limitations: Limited World Englishes
Roach, P. (2020). English phonetics and phonology (5th ed.). Cambridge University Press. [EN]
```

**Inline**: "TRAP-BATH split [Ref: L2] affects /æ/ → /ɑː/ in RP [Ref: G5]"
