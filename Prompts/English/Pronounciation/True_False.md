# True/False Statements

Generate 24-40 intermediate/advanced true/false statements on English pronunciation with authoritative evidence.

**Domain**: English pronunciation (AmE, BrE, Indian, other major accents) — phonetics, phonology, suprasegmentals, accent features.

**Requirements**: Objectively verifiable via authoritative linguistic sources. Accent comparisons require clear context and reference points.

---

## Workflow

### 1. Plan (MECE Coverage)

**6-8 topic clusters** (mutually exclusive, collectively exhaustive):
- 4-6 statements/cluster (total: 24-40)
- Difficulty: 15% Foundational / 45% Intermediate / 40% Advanced

**Essential Clusters**:
1. **Vowel Systems**: monophthongs, diphthongs, accent comparisons
2. **Consonants**: problematic sounds, allophonic variation, flapping, rhoticity
3. **Word Stress**: patterns, compound stress, stress shift
4. **Connected Speech**: linking, assimilation, elision, weak forms
5. **Intonation & Rhythm**: stress-timed vs syllable-timed, pitch patterns
6. **American vs British**: systematic differences, lexical sets
7. **Indian English & Other Accents**: retroflex, aspiration, rhythm, vowel mergers
8. **Problematic Sounds**: minimal pairs, L1 interference

**Verify**: Total 24-40? Difficulty ≈15/45/40? Non-overlapping? Complete coverage?

### 2. Collect References (Credibility)

**Minimums**: G≥15 | AC≥4 | L≥8 | APA≥15

**Source Hierarchy**:
1. Phonetic authorities (IPA, Cambridge/Oxford dictionaries, Wells' Lexical Sets)
2. Peer-reviewed journals (Journal of Phonetics, Language and Speech)
3. Audio corpora (IDEA, Speech Accent Archive, phonetic databases)
4. Authoritative textbooks (Gimson, Roach, Ladefoged, Wells)
5. Standard references (CEPD, LPD)

**Exclude**: Non-academic blogs, unverified apps, crowdsourced dictionaries without expert validation

**Quality Gates**:
- Recency: ≥40% last 5yr
- Diversity: ≥3 types (dictionaries, textbooks, journals, corpora), no single >30%
- Language: ~70% EN / ~20% ZH / ~10% other (tag all)
- Links: All accessible/archived (audio samples persistent)

**IDs**: G1-Gn (Glossary) | AC1-ACn (Audio/Corpus) | L1-Ln (Literature) | A1-An (APA)

**Verify**: Minimums met? Language ≈70/20/10? Recency ≥40%? Types ≥3? Links valid?

### 3. Write Statements (Clarity & Logic)

**Statement** (≤2 lines, declarative, unambiguous):
- No double negatives
- Define phonetic terms in Glossary (use IPA where appropriate)
- Specify accent context when relevant
- Assign: Difficulty + Answer (A=True / B=False)

**Rationale** (1-2 sentences):
- Explain WHY (phonetic principle, not just description)
- Cite ≥1 `[Ref: ID]` (≥2 for accent comparisons/contested claims)
- Specify register/regional variation/phonological environment when context-dependent
- Acknowledge accent variation/sociolinguistic factors/evolving norms when controversial

**Verify every 5**: ≤2 lines? No double negatives? Unambiguous? Accent context clear? IPA defined? Rationale cited?

### 4. Balance (Fairness)

- True/False ratio: 45-55%
- No patterns (avoid TFTFTF, clustering)

**Verify**: Ratio 45-55%? No sequences? Distributed across topics/difficulty?

### 5. Compile References

**Glossary**: `**G#: Term/IPA**: Definition [Lang]`  
**Audio/Corpus**: `**AC#: Name** ([lang])` — Type, Coverage, Access URL  
**Literature**: `**L#: Title** (Year) ([lang])` — Findings, Methodology, Scope  
**APA**: APA 7th + language tags, grouped by language, persistent links

**Verify**: All `[Ref: ID]` resolve? Required fields present? No unused refs? Links valid?

### 6. Validate (13 Checks)

Execute sequentially, fix failures, re-validate until all PASS:

1. **Floors**: G≥15, AC≥4, L≥8, APA≥15, S:24-40, Difficulty ≈15/45/40
2. **Citations**: ≥70% with ≥1 cite, ≥30% with ≥2
3. **Language**: EN 60-80%, ZH 15-25%, Other 5-15%
4. **Recency**: ≥40% last 5yr
5. **Diversity**: ≥3 types, no single >30%
6. **Links**: 100% accessible/archived
7. **Cross-refs**: All `[Ref: ID]` resolve
8. **Clarity**: ≤2 lines, unambiguous, accent context specified
9. **Negatives**: No double negatives
10. **Rationales**: 1-2 sentences + citation, principle explained
11. **Context**: ≥80% specify accent/register/environment
12. **Fairness**: ≥90% balanced (no single accent bias)
13. **IPA/Terms**: 100% defined in Glossary
