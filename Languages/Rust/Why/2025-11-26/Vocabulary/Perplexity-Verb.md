# Verb Vocabulary - Perplexity

1. **Verb**: compile (intransitive/transitive)
   **Meaning**: 
   - *(Programming)* To translate source code into executable machine code
   - To collect and arrange information into a list or publication
   
   **Synonyms**: build, translate, assemble, generate, produce
   
   **Antonyms**: interpret, decompile, execute
   
   **When to Use**: Programming contexts; document preparation contexts
   
   **When NOT to Use**: When referring to running programs (use "execute")
   
   **Common Phrases**: 
   - compile code
   - compile successfully
   - fail to compile
   
   **Root Analysis**: com (together) + pile (heap)
   
   **Etymology**: Latin → compilare (plunder, gather together)
   
   **Examples [Casual]**: 
   - The code won't compile - there's an error somewhere.
   - I'm compiling a list of resources for the project.
   
   **Examples [Formal]**: 
   - Rust compiles async functions into state machines for zero-cost abstraction.
   - The compiler compiles generic code into specialized implementations.

1. **Verb**: enforce (transitive)
   **Meaning**: 
   - To compel observance of or compliance with; cause to take effect
   - To impose or apply with authority
   
   **Synonyms**: impose, implement, apply, administer, execute
   
   **Antonyms**: waive, relax, ignore, abandon
   
   **When to Use**: Legal, technical, and policy contexts requiring compliance
   
   **When NOT to Use**: Casual requests or suggestions
   
   **Common Phrases**: 
   - enforce rules
   - enforce constraints
   - strictly enforce
   
   **Root Analysis**: en (in) + force
   
   **Etymology**: Old French → enforcier (strengthen)
   
   **Examples [Casual]**: 
   - They need to enforce the parking rules more strictly.
   - Parents should enforce consistent bedtimes.
   
   **Examples [Formal]**: 
   - The borrow checker enforces aliasing XOR mutation rules at compile time.
   - Rust enforces memory safety through strict compile-time verification.

1. **Verb**: reject (transitive)
   **Meaning**: 
   - To refuse to accept, consider, or submit to
   - *(Compiler)* To refuse to compile code due to errors
   
   **Synonyms**: refuse, decline, deny, dismiss, rebuff
   
   **Antonyms**: accept, approve, embrace, adopt
   
   **When to Use**: Decision-making and technical contexts
   
   **When NOT to Use**: Casual declines (can sound harsh)
   
   **Common Phrases**: 
   - reject code
   - compiler rejects
   - reject proposal
   
   **Root Analysis**: re (back) + jacere (throw)
   
   **Etymology**: Latin → rejectare (throw back)
   
   **Examples [Casual]**: 
   - The application was rejected for incomplete information.
   - Don't reject the idea without considering it first.
   
   **Examples [Formal]**: 
   - The borrow checker rejects code that might violate safety invariants.
   - Rust intentionally rejects some valid programs to ensure comprehensive safety.

1. **Verb**: defer (transitive/intransitive)
   **Meaning**: 
   - To postpone or delay action; put off to a later time
   - To shift responsibility or execution to another time or entity
   
   **Synonyms**: postpone, delay, shelve, table, put off
   
   **Antonyms**: expedite, advance, hasten, accelerate
   
   **When to Use**: Professional contexts describing postponement
   
   **When NOT to Use**: When immediate action is critical
   
   **Common Phrases**: 
   - defer to runtime
   - defer decision
   - defer execution
   
   **Root Analysis**: de (from) + ferre (carry)
   
   **Etymology**: Latin → differre (carry apart, postpone)
   
   **Examples [Casual]**: 
   - Let's defer this discussion until next week.
   - I'll defer to your expertise on this matter.
   
   **Examples [Formal]**: 
   - Other languages defer safety checks to runtime whereas Rust performs them at compile time.
   - Garbage collectors defer memory management to runtime with non-deterministic timing.

1. **Verb**: prohibit (transitive)
   **Meaning**: 
   - To formally forbid something by law, rule, or authority
   - To prevent or make impossible
   
   **Synonyms**: forbid, ban, prevent, disallow, preclude
   
   **Antonyms**: allow, permit, authorize, enable
   
   **When to Use**: Legal and technical contexts establishing restrictions
   
   **When NOT to Use**: Casual contexts where "not allow" is clearer
   
   **Common Phrases**: 
   - prohibit access
   - strictly prohibited
   - prohibit simultaneous
   
   **Root Analysis**: pro (before) + habere (hold)
   
   **Etymology**: Latin → prohibere (hold back, restrain)
   
   **Examples [Casual]**: 
   - The rules prohibit smoking in the building.
   - Speed limits prohibit driving over 65 mph here.
   
   **Examples [Formal]**: 
   - Rust prohibits simultaneous mutable and immutable references to prevent data races.
   - The type system prohibits operations that could violate memory safety.

1. **Verb**: expose (transitive)
   **Meaning**: 
   - To make visible or reveal; uncover
   - *(Programming)* To make accessible or visible to other code
   
   **Synonyms**: reveal, uncover, disclose, show, make visible
   
   **Antonyms**: hide, conceal, encapsulate, obscure
   
   **When to Use**: Technical and investigative contexts
   
   **When NOT to Use**: Photography contexts (different technical meaning)
   
   **Common Phrases**: 
   - expose API
   - expose complexity
   - expose errors
   
   **Root Analysis**: ex (out) + ponere (place)
   
   **Etymology**: Latin → exponere (set forth)
   
   **Examples [Casual]**: 
   - The investigation exposed serious problems.
   - Don't expose your password to anyone.
   
   **Examples [Formal]**: 
   - The runtime-agnostic design exposes complexity that other languages hide behind runtimes.
   - Async programming exposes lifetime constraints explicitly in type signatures.

1. **Verb**: mandate (transitive)
   **Meaning**: 
   - To make something mandatory; require by authority
   - To officially require or demand
   
   **Synonyms**: require, demand, necessitate, ordain, decree
   
   **Antonyms**: suggest, recommend, permit, allow
   
   **When to Use**: Formal contexts establishing requirements
   
   **When NOT to Use**: Casual suggestions or preferences
   
   **Common Phrases**: 
   - mandate compliance
   - mandate use of
   - legally mandated
   
   **Root Analysis**: from Latin mandatum
   
   **Etymology**: Latin → mandatum (command, order)
   
   **Examples [Casual]**: 
   - The new policy mandates weekly reports.
   - Safety regulations mandate wearing helmets.
   
   **Examples [Formal]**: 
   - As a systems language, Rust cannot mandate a one-size-fits-all async runtime.
   - The language design does not mandate built-in runtime support for embedded compatibility.

1. **Verb**: optimize (transitive)
   **Meaning**: 
   - To make as effective, perfect, or functional as possible
   - *(Computing)* To modify code for maximum efficiency
   
   **Synonyms**: improve, refine, enhance, perfect, streamline
   
   **Antonyms**: degrade, worsen, impair
   
   **When to Use**: Technical and process improvement contexts
   
   **When NOT to Use**: Contexts requiring "maximize" (different meaning)
   
   **Common Phrases**: 
   - optimize performance
   - optimize for speed
   - highly optimized
   
   **Root Analysis**: optimal + -ize
   
   **Etymology**: Latin → optimus (best) + verb suffix
   
   **Examples [Casual]**: 
   - We need to optimize the workflow to save time.
   - The app was optimized for mobile devices.
   
   **Examples [Formal]**: 
   - LLVM aggressively optimizes intermediate representation for maximum performance.
   - Developers must optimize domain-specific implementations while maintaining safety.

1. **Verb**: traverse (transitive)
   **Meaning**: 
   - To travel across or through
   - *(Computing)* To visit all nodes or elements in a data structure
   
   **Synonyms**: cross, navigate, move through, iterate, scan
   
   **Antonyms**: avoid, bypass, skip
   
   **When to Use**: Technical and geographical contexts
   
   **When NOT to Use**: Simple movement (use "go through")
   
   **Common Phrases**: 
   - traverse graph
   - traverse tree
   - traverse array
   
   **Root Analysis**: trans (across) + versus (turned)
   
   **Etymology**: Latin → transversare (cross)
   
   **Examples [Casual]**: 
   - We traversed the mountain trail in three hours.
   - The explorer traversed the entire continent.
   
   **Examples [Formal]**: 
   - Irregular parallelism patterns like graph traversal require careful synchronization.
   - The algorithm traverses the dependency tree to resolve conflicts.

1. **Verb**: instantiate (transitive)
   **Meaning**: 
   - *(Programming)* To create an instance of a class or type
   - To represent something abstract by a concrete example
   
   **Synonyms**: create, realize, implement, generate, produce
   
   **Antonyms**: destroy, abstract, generalize
   
   **When to Use**: Programming and philosophical contexts
   
   **When NOT to Use**: Casual contexts where "create" suffices
   
   **Common Phrases**: 
   - instantiate object
   - instantiate type
   - instantiate template
   
   **Root Analysis**: instant + -iate
   
   **Etymology**: Medieval Latin → instantiare (be present)
   
   **Examples [Casual]**: 
   - You need to instantiate the class before using it.
   - Let's instantiate this idea with a concrete example.
   
   **Examples [Formal]**: 
   - Monomorphization instantiates generic functions for each concrete type used.
   - The compiler must instantiate trait implementations for specific types.
